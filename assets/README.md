# Assets 目录

此目录用于存放应用的静态资源文件。

## 所需文件（可选）

以下文件是 Expo 项目的标准图标，如果不存在会使用默认图标：

- `icon.png` - 应用图标（1024x1024px）
- `splash.png` - 启动屏幕（1242x2436px）
- `adaptive-icon.png` - Android 自适应图标（1024x1024px）
- `favicon.png` - Web 图标（48x48px）

## 注意

如果你想快速测试应用，这些图标文件是**可选的**。Expo 会使用默认占位图标。

你可以稍后添加这些文件来自定义应用外观。
